1. Who We Are

Researchtoolslab (researchtoolslab.github.io) is a free online tools website providing research, statistical, financial, and creator calculators. This privacy policy explains how we handle (or rather, do not handle) your personal information.

2. Data We Collect

We collect no personal data. Researchtoolslab does not collect names, email addresses, IP addresses (beyond what your browser sends to any web server), payment information, or any other identifying information.

All calculator inputs ‚Äî loan amounts, research parameters, earnings estimates ‚Äî are processed entirely within your browser using JavaScript. No input data is transmitted to any server, database, or third party.

3. Cookies & Local Storage

We do not use advertising cookies, analytics cookies, or tracking pixels of any kind. The PDF Invoice Generator tool uses your browser's localStorage solely to remember your last invoice number across sessions. This data never leaves your device.

You can clear this at any time through your browser's settings (Settings ‚Üí Privacy ‚Üí Clear site data).

4. Third-Party Services

GitHub Pages: This website is hosted on GitHub Pages. GitHub may collect server access logs (IP addresses, browser type, timestamps) as part of standard hosting infrastructure. See GitHub's privacy policy for details.

Google Fonts: We load fonts (Fraunces, DM Sans, DM Mono) from Google Fonts CDN. This means your browser makes a request to Google's servers to download these font files. Google may log this request. See Google's privacy policy for details. If you prefer, you can block Google Fonts via a browser extension ‚Äî the site remains fully functional using system fonts.

jsPDF (PDF Generator only): The PDF Invoice Generator loads the jsPDF library from Cloudflare's CDN. Cloudflare may log requests to their CDN. The PDF generation itself runs entirely in your browser.

5. Analytics

We do not currently use any analytics service (Google Analytics, Plausible, Fathom, or similar). We have no visibility into individual user behaviour, page views, or sessions beyond what GitHub Pages' server logs contain.

6. Children's Privacy

Researchtoolslab does not knowingly collect any information from children under 13. As we collect no personal data from any user, this requirement is satisfied by design.

7. Changes to This Policy

If this policy changes (for example, if we add analytics or advertising in the future), we will update the "Last updated" date above and add a summary of changes. We will never retroactively change how previously submitted data is used because we do not collect any.

8. Contact

If you have questions about this privacy policy, you can open an issue on our GitHub repository at github.com/researchtoolslab.
